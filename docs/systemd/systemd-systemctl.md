### Analyzing the system state:

#### System status:
```
systemctl status
```
#### List running units:
```
systemctl 
systemctl list-units

```

#### List failed units:

```
systemctl --failed

```

#### List installed unit files:

```

systemctl list-unit-files

```

#### cgroup slice, memory and parent for a PID
```
systemctl satus pid

```
